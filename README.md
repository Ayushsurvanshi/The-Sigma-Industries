## About

Sigma Industries is a leading manufacturer and supplier of superior quality aluminum die-casting ladles and foundry tools, established in 2003 by visionary entrepreneur Vijay Survanshi. Based in Kolhapur, Maharashtra, India, the company has established itself as a trusted partner for precision-engineered solutions in the aluminum die-casting sector with over 22 years of industry experience.

**Key Information:**
- **Established:** 2003
- **Founder & Managing Director:** Vijay Survanshi
- **Headquarters:** Kolhapur, Maharashtra, India
- **Industry Experience:** 22+ Years
- **Service Areas:** Maharashtra, Gujarat, Karnataka, Goa

**Mission:** To deliver durable, precision-engineered ladles that enhance efficiency and safety in aluminum die-casting processes, backed by our unwavering commitment to customer satisfaction.

**Vision:** To be the preferred global supplier of innovative foundry solutions, setting industry standards for quality, reliability, and customer service in the aluminum die-casting industry.

## Installation

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- Internet connection

### Setup Instructions

```bash
# Clone the repository
git clone [repository-url]
cd sigma-industries-website

# Open index.html in your browser
# Or use a local server
python -m http.server 8000
# Then visit http://localhost:8000
```

### Deployment
- Upload all files to your web server
- Ensure proper file permissions
- Configure domain settings
- Test all functionality
